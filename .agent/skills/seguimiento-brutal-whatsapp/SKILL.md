# Seguimiento Brutal - WhatsApp

Esta habilidad genera secuencias de mensajes de seguimiento para WhatsApp que convierten. El objetivo es recuperar la atención del lead sin parecer desesperado ni robótico.

## Input Requerido
1.  **Nombre del lead**: (Opcional, para personalizar).
2.  **Rubro y Contexto**: Qué vendes y cuál es la situación.
3.  **Etapa**: ¿Es un lead frío, post-llamada, o ya tiene propuesta?
4.  **Objeción Principal**: "Lo pienso", "No tengo tiempo", "No tengo dinero", "Miedo", o "Visto y no responde".
5.  **CTA Deseado**: Agendar, responder preguntas, aprobar.

## Reglas de Oro (ANTI-ROBOT)
1.  **Cero "ChatGPT-ismo"**: Prohibido usar "Espero que estés bien", "Quería consultarte", "En referencia a...". Sé natural. Escribe como escribes a un amigo, pero con respeto profesional.
2.  **Brevedad**: Mensajes de 1-3 líneas máximo. WhatsApp se lee rápido.
3.  **Siempre Preguntar**: NUNCA envíes un mensaje que no termine en una pregunta o CTA claro.
4.  **No Jerga**: Habla claro.

## Estructura de Salida Obligatoria

Genera la respuesta en bloques de código o texto fácil de copiar.

### 1. Secuencia Temporal (La Base)
Genera mensajes específicos para estos momentos, adaptados a la objeción:
*   **24h después**: (Corto, casual).
*   **48h después**: (Aportando valor o recordando el dolor).
*   **72h después**: (Directo al punto).
*   **7 días después**: (Reactiva la conversación).

### 2. Kit de Resurrección (4 Mensajes "Revive")
Mensajes atemporales para leads muertos hace semanas.
*   *Idea 1*: Novedad/Update.
*   *Idea 2*: "¿Sigues ahí?".
*   *Idea 3*: meme/apilar valor (descrito en texto).
*   *Idea 4*: "Estaba pensando en tu caso...".

### 3. Variaciones de Estilo (3 Opciones por cada uno)
Para que el usuario elija su "voz":
*   **Opción A (Amable/Suave)**: Empático pero firme.
*   **Opción B (Directo/Al hueso)**: Para gente ocupada.
*   **Opción C (Elegante "Break-up")**: Último intento, retirando la oferta para generar escasez.

### 4. Preguntas de Calificación (6 Pack)
Preguntas de sí/no o respuesta corta para destrabar el chat.
*   (Ej: "¿Es un tema de tiempos o de presupuesto?", "¿Ya resolviste X?").

### 5. Cierres "Take-away" (3 Opciones)
Cierres donde le dices al cliente que está bien si no compran, quitando presión.
*   (Ej: "Si no es el momento, avísame y no te molesto más. ¿Te parece?").
